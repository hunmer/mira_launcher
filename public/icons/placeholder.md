# 应用图标目录

请在此目录放置应用图标文件：

- chrome.png - Google Chrome 图标
- vscode.png - Visual Studio Code 图标
- notepad.png - 记事本图标
- figma.png - Figma 图标
- slack.png - Slack 图标
- spotify.png - Spotify 图标
- notion.png - Notion 图标
- postman.png - Postman 图标

图标尺寸建议：
- 最小尺寸：32x32px
- 推荐尺寸：64x64px 或 128x128px
- 最大尺寸：200x200px
- 格式：PNG（支持透明背景）
